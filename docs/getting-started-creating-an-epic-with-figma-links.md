# Creating an Epic with Figma Links

As part of the story writing tools ([analyze-feature-scope](../server/providers/combined/tools/analyze-feature-scope/README.md), [write-shell-stories](../server/providers/combined/tools/writing-shell-stories/README.md), and [write-next-story](../server/providers/combined/tools/write-next-story/README.md)), you will need an epic with figma links. This page walks through how to create one so story writing will be successful.
